# Input

```jsx
import { Input } from "ui"

<Input
    className="w-full"
    id="test_input"
    placeholder="Introduce tu nombre"
    value={name}
    onChange={(e) =>setName(e.target.value)}
/>
```

<Example />

### Parámetros

- `styleType`: Estilo del componente. Por defecto `primary`.
- `type`: Tipo de input. Por defecto `text`.
- `className`: Clase del componente.
- `onChange`: Función que se ejecuta al cambiar el valor del input.
- `onBlur`: Función que se ejecuta al perder el foco del input.
- `onFocus`: Función que se ejecuta al tener el foco del input.
- `value`: Valor del input.
- `placeholder`: Texto que se muestra en el input.
- `disabled`: Si el input está deshabilitado.
- `readOnly`: Si el input es solo lectura.
- `required`: Si el input es obligatorio.
- `autoComplete`: Si el input debe autocompletar.
